# Installing on Windows

## Manual Python installer:

If the 'all users' install is chosen Python the default directory will be:

* C:/Python3x

When installed for an individual user the location wioll be:

* C:\Users\<user>\AppData\Local\Programs\Python\Python3x

Note the 'Programs' part.

## Visual Studio
[Microsoft Visual Studio](https://visualstudio.microsoft.com/downloads/) uses special locations when managing Python installations, similar to:

* C:\Program Files (x86)\Microsoft Visual Studio\Shared\Python37_64

See this [SO answer](https://stackoverflow.com/questions/52021815/where-visual-studio-install-python) for more information.
